grep（global search regular expression(RE) and print out the line，全面搜索正则表达式并把行打印出来）是一种强大的文本搜索工具，它能使用正则表达式搜索文本，并把匹配的行打印出来。


## grep命令：

grep [options] ‘pattern’ filename

#### 表达符集

```
^ 锚定行的开始： grep ‘^a’ aa.txt  
$锚定行的结束：grep ‘$a’ aa.txt
.匹配任何一个非换行符（’\n’），：grep ‘a.b’ 匹配任何包含a*b字符的行
匹配以a*开头的行：grep ‘^a.’ aa.txt
	➜  linux grep '^t.' aa.txt
	two
	three
*匹配0个和多个先前字符：grep ‘a*b’ aa.txt 匹配有一个或者多个a后面是b字符的行;grep ‘ab*’ aa.txt，匹配a后面有0个或者多个b的字符串的行；.*代表匹配任意字符且不限长度。todo：grep ‘a*’ aa.txt 会返回所有行?
[] 匹配一个指定范围内的字符:grep '[rn]e' aa.txt 匹配re或者ne的行；grep '[rn]*e' aa.txt匹配有一个躲着多个r或者n后面是e的字符串的行
[^] 匹配一个不再指定范围内的字符。：grep ‘[^a]’ aa.txt 匹配不包含a的字符的行；匹配不包含grep ‘[^a-z]’ aa.txt，匹配不包含字符的行；grep ‘^$’ aa.txt 匹配空行。
\(..\) 标记匹配字符，grep '\(one\)' aa.txt : one被标记为1 。todo:如何使用呢？
\< 锚定单词的开始。 grep '\<on' aa.txt 匹配以on开头的行。todo：等于^?
\> 毛低昂单词的结束。 grep 'on>\’ aa.txt 匹配以on结束的行。todo： 等于$?
x\{m,n\} 重复字母x次。grep 'e\{2,\}' aa.txt 匹配重复字符x两次的字符串的行。m,n的语法是前缀和后缀，代表匹配次数区间，缺少参数代表开区间，只有一个值就是准确次数，比如x\{3\}，没有逗号，就是3次。
\w 匹配字母和数字字符，相当于[A-Za-z0-9]。grep '\we' aa.txt 匹配任意字母或者数字连接e的字符串的行
\W 匹配一个或者多个非英文字符，比如逗号,句号,汉字等。grep '\W\.' aa.txt 匹配所有非英文字符与.连接的字符串的行
\b 单词锁定符。grep '\bone\b' aa.txt 匹配含有one单词的那一行。todo：不能匹配单词中的部分？
\+ 匹配一个或者多个先前字符（注意与.的区别，0个和1个）.grep 'e\+e' aa.txt 匹配至少两个e相连的字符串的行；grep 'e*e' aa.txt 匹配一个或者多个e相连的字符串的行
\? 匹配0个或者1个先前字符。grep 'a\?' aa.txt 匹配0个或者1个a的字符串的行。
a\|b\|c 匹配a或者b或者c。grep 'a\|ee' aa.txt 匹配a或者ee，不是ae或者ee，|运算符优先级低，字符串连接优先级高
\(\) 分组符号 grep 'e\(\(b\)\?\)\|a' aa.txt 匹配e连接0个或者1个b的字符串或者a字符串的行。可以按照数学表达式的规律使用。
```
#### 参数列表：

```
-v 显示包含匹配内容的所有行

-l 显示包含匹配的文件名：grep -l 'test' aa.txt ./test/test.txt 查找区间可以是多个参数或者一个参数的同级目录下的所有文件

-I 忽略大小写

-n 显示匹配的行号

-c 搜寻结果匹配的次数

-w 按照单词搜索 匹配的内容是一个单词

-r 逐级遍历目录

-A: 显示匹配行及前面多少行, 如: -A3, 则表示显示匹配行及前3行  todo：测试相反？

-B: 显示匹配行及后面多少行, 如: -B3, 则表示显示匹配行及后3行   todo：测试相反？
-C: 显示匹配行前后多少行,   如: -C3, 则表示显示批量行前后3行   

--color: 匹配到的内容高亮显示
--include: 指定匹配的文件类型
--exclude: 过滤不需要匹配的文件类型

```
